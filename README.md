# Dronest — The Infinite Loop

Product website for **Dronest**: autonomous drone power infrastructure —
NMC smart batteries, autonomous cooled Sky-Dock charging, and a
hop-and-charge network for persistent fleets.

Live site: https://dronesttt.github.io/dronest-site/

Static single-page site — no build step. To preview locally, open
`index.html` or serve the folder:

```
python -m http.server 8642
```

Contact: support@dronest.in
